# Welcome to My Workspace

Hi — I'm a software engineer from Indonesia who loves building things.
Backend when the API needs saving, frontend when the UI needs rescuing,
and horizontal on the couch the rest of the time — which, honestly,
is where half my best ideas come from.

This portfolio runs like my editor because that's the only environment
I actually trust. Open a file from the sidebar, or hit `Ctrl+K` to jump
anywhere. Recruiters: the important stuff is one click away.

## Shortcuts

- `Ctrl+K` / `Cmd+K` — Command palette
- Right-click anywhere — also opens the command palette
- Click any file in the **Explorer** sidebar on the left
- Close tabs with the `×` — last tab always stays open
